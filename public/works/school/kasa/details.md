# Kasa

## Project Overview
Kasa is a professional-grade web application developed as part of my "Web Developer" certification at OpenClassrooms. The project focuses on creating a high-performance, responsive rental property platform. This project was a key milestone in demonstrating my proficiency with the React ecosystem, and it was successfully validated with honors during my final defense.

## My Role: Frontend Developer
I built the entire application from scratch based on a provided design, focusing on modular architecture and modern styling techniques:
* **Component-Based Architecture:** Developed the site using React to create reusable UI components, ensuring a clean and maintainable codebase.
* **Advanced Routing:** Implemented React Router for seamless client-side navigation between the home gallery and detailed property views.
* **Interactive UI Elements:** Created custom React components for advanced features like image carousels, accordion-style "collapsibles" for descriptions and amenities, and dynamic star-rating systems.
* **Modern Styling:** Utilized SCSS for structured, maintainable styles, ensuring full responsiveness across mobile, tablet, and desktop devices.
* **Data Management:** Managed and displayed property data dynamically from a JSON source, simulating a real-world API integration.
