# Outside

## Project Overview
Outside is a third-person stealth-action game developed as a final year project during my Gameplay Programming studies. Created over a 2-month period, the game follows a child with telekinetic powers attempting to escape a high-security laboratory. I served as the Lead Programmer for a team of three developers and three artists.

## My Role: Lead Gameplay Programmer
As the programming lead, I was responsible for the technical vision and the implementation of the core interaction systems entirely within Blueprints:
* **Telekinesis Mechanics:** Engineered the physics-based interaction system allowing players to manipulate, lift, and throw environmental objects to solve puzzles and distract guards.
* **Character Controller:** Developed the player's movement set, ensuring fluid controls that felt responsive and intuitive for a stealth-focused experience.
* **In-Engine Cinematics:** Created a custom system to handle seamless transitions between gameplay and in-game narrative sequences.
* **Technical Leadership:** Orchestrated the programming team’s workflow, handled task distribution, and ensured code consistency across the project.
* **Audio Integration:** Implemented the complete soundscape, including environmental SFX, telekinetic feedback sounds, and the musical score.
