# Rebel Cops

## Project Overview
Rebel Cops was my debut professional project in the video game industry. A tactical spin-off in the "This Is the Police" series, I contributed to the Unity-based development with a heavy focus on UI/UX design, implementation, and workflow automation.

## My Role: UI & Tools Developer
I played a key role in bridging the gap between design and technical implementation, ensuring the interface was both ergonomic and performant. My contributions included:
* **UI/UX Design & Implementation:** Participated in design meetings to propose intuitive interface solutions and subsequently implemented the entire UI system within Unity.
* **Custom Editor Tooling:** Developed a specialized Unity Editor tool featuring a custom node graph. This tool automated the management of mobile asset import settings based on their parent directory, significantly streamlining the pipeline.
* **Optimization & Polishing:** Conducted extensive bug fixing and performance optimization to ensure a stable experience across target platforms.
* **Mobile Workflow Automation:** Engineered logic to handle complex asset configurations automatically, reducing manual errors during the porting process.

## Challenges & Solutions

### 1. Navigating a First Professional Industry Project
**Problem:** Entering the professional game industry for the first time presented a steep learning curve regarding production standards and industry-specific workflows.
**Solution:** I adopted a proactive learning approach, frequently asking targeted questions and requesting regular code reviews from my senior colleagues. This open feedback loop allowed me to align my coding style with professional standards rapidly.

### 2. Large and Complex Existing Codebase
**Problem:** The project featured an extensive, multi-layered codebase that was initially difficult to navigate and modify without unintended side effects.
**Solution:** I dedicated a structured period to "mapping out" the architecture. By experimenting with isolated features and tracing data flows, I gained a comprehensive understanding of the project's logic, which enabled me to contribute safely and efficiently.

### 3. Inefficient Asset Pipeline for Mobile
**Problem:** The project contained a massive volume of assets that were not optimized for mobile hardware, and manual configuration was leading to inconsistencies and performance bottlenecks.
**Solution:** I developed a custom Unity Editor tool that automatically applied optimized import settings based on an asset's directory location. I also automated the creation of Asset Bundles, ensuring that the mobile build remained lightweight and performant while significantly reducing manual overhead for the art team.
