# Hytrade

## Project Overview
Hytrade is a specialized trading modification for the game Hytale, developed immediately upon the game's release. Created in an environment with zero official documentation, the project required deep technical analysis of the game's architecture to provide a secure, server-mediated exchange system for MMORPG environments.

## My Role: Lead Developer & Reverse Engineer
I spearheaded the technical development and design of the mod, focusing on security and cross-mod compatibility:
* **Reverse Engineering:** Decompiled the game's `.jar` server files to map out internal logic and API structures in the absence of official developer tools.
* **Secure Transaction Logic:** Engineered a robust "Middleman" system where the server acts as a secure intermediary to prevent item loss or player fraud during trades.
* **Ecosystem Integration:** Developed and maintained compatibility layers for various third-party economy mods, allowing players to trade both physical items and digital currency seamlessly.
* **Graphic Design:** Created all custom UI icons and visual assets within Figma to match the game's native aesthetic.
* **Project Maintenance:** Managed the mod's lifecycle, reaching over 1,500 downloads while providing consistent updates and bug fixes.
