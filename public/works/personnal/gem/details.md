# Gem

## Project Overview
Gem is an experimental AI-assisted note-taking application developed in a single week. The project served as a research platform to explore "Vibecoding" workflows and optimize human-AI collaboration. Beyond traditional note-taking, Gem features local AI integration for contextual search and automated summaries, moving away from simple keyword matching.

## My Role: Lead Developer & Workflow Architect
I acted as both the UI designer and the orchestrator of an AI-driven development pipeline. My work involved:
* **UI/UX Strategy:** Manually designing initial mockups to serve as a visual "source of truth" for the AI, ensuring a consistent and polished aesthetic.
* **Vibecoding Guidelines:** Developed a rigorous set of guidelines to optimize AI context. This included pre-task and post-task instructions that allow different AI agents to understand the project’s UI/UX standards and general architecture without scanning the entire codebase.
* **Process Automation:** Established a real-time modification tracking system (ordered by most recent changes) and a dynamic TODO list to maintain seamless continuity between AI-assisted coding sessions.
* **Full-Stack Implementation:** Built the application using Vue.js and Prisma ORM, integrating local AI models via OpenRouter to handle complex NLP tasks like summarization and semantic search.

* **Contextual Search:** AI-powered search that understands user intent and context rather than just matching keywords.

* **Optimized AI Workflow:** A custom framework for AI prompting, focusing on input showcases and modular context management.
